This backend provides APIs for authentication, file uploads, and data management using Express, Mongoose, and JWT.

## Installation

cd backend
npm install

Environment Variables

Create a .env file:

PORT=5000
MONGO_URI=your_mongo_uri
JWT_SECRET=your_jwt_secret

Run the Server

Development
npm run dev

Production
npm start
