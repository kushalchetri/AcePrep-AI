This is the frontend for the Interview Prep AI project.  
It provides a clean UI for generating questions, reviewing answers, and interacting with AI using the backend APIs.

---

## Features

- React 18 (SPA)
- Vite for fast development + builds
- Tailwind CSS v4
- Axios for API calls
- React Router for navigation
- React Markdown + GFM
- Syntax highlighting via Prism
- Framer Motion animations
- Toast notifications

---

## Installation

cd frontend
npm install

Development
npm run dev
Build for Production
npm run build
